# HabitFlow – Habit Tracker

HabitFlow is a simple web-based habit tracking application that allows users to track daily habits, view progress through a calendar interface, and manage personal goals.

The project was built as a personal development project to strengthen my experience with backend development, database integration, and building small web applications.

## Features

- User registration and login system
- Create and manage personal habits
- Calendar view for tracking daily completion
- Toggle habits as completed or incomplete
- Simple and responsive interface
- Data export functionality

## Tech Stack

- PHP
- MySQL
- HTML / CSS
- JavaScript
- XAMPP for local development
